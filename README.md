# Sample AWS Web Page

このリポジトリは、AWSの基本操作経験を示すサンプルとして作成しました。

## やったこと

- AWS EC2でApacheサーバーを起動し、HTTPリクエストに応答するWebページを作成
- index.htmlのパーミッションや所有権を調整して正常に閲覧可能に
- Apacheのアクセスログを確認し、403/404エラーの原因を特定・修正
- GitHub Pagesに静的サイトとして公開し、ブラウザでアクセス可能に

## 学んだこと

- EC2インスタンスの基本操作（起動・停止・接続）
- HTTPリクエストの流れとApacheの仕組み
- ファイル権限（chmod）の調整と影響
- GitHubリポジトリ管理とGitHub Pagesでの公開手順
